About ikuda.net and it's services / subdomains


ikuda.net

ikuda.net is registered on Spaceship and the DNS is handled by Cloudflare, Zoho Mail handles emails. ikuda.net does not use site generators, I write the HTML and CSS by hand onto a GitLab repo (Note: only the pro pages use bootstrap). When a new commit is made, the repo is automatically deployed onto Cloudflare's network by Cloudflare Pages.


archive.ikuda.net

The ikuda archive operates the same way as ikuda.net; GitLab repo deployed through Cloudflare Pages. Contains old pages. (eg. blog posts, deprecated pages)


r2.ikuda.net

r2.ikuda.net is a Cloudflare R2 storage bucket behind Cloudflare Access.


eu-external.ikuda.net, eu-redistributed.ikuda.net

These R2 buckets are both not behind Access. eu-external is used for storing static assets for my websites outside of ikuda.net. (eg. Neocities) eu-redistributed is used for storing the redistributed resources in mental health. Both buckets are located in the European Union, hence the eu- prefix. For DMCA info regarding eu-redistributed, see this page.


woem.workers.ikuda.net

A very basic Cloudflare Worker; see the code below. 

export default {
  async fetch(request, env, ctx) {
    return new Response('woem!');
  },
};

About old services


enby.me

enby.me was ikuda.net before I registered ikuda.net. In the start, it used Netlify for website deployments and DNS. Slowly, the domain was completely migrated to Cloudflare.


gay.enby.me, comfy.enby.me

gay.enby.me was an Akkoma instance, and comfy.enby.me was a GoToSocial instance. Both ran on Digitalocean droplets.
